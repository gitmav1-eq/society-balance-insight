# SOCIETY.EXE

> **"Zoom out. See the consequences clearly."**

An interactive educational web platform that helps users understand the collective impact of individual financial behaviors on society. SOCIETY.EXE visualizes how normalized financial choices—when scaled across millions—create systemic pressures and shape economic patterns.

![SOCIETY.EXE Preview](https://via.placeholder.com/1200x630/1a1a2e/6366f1?text=SOCIETY.EXE+-+Financial+Behavior+Simulator)

## 🌟 Overview

SOCIETY.EXE transforms abstract financial concepts into tangible, interactive experiences. By simulating individual behaviors at scale, users gain insight into the invisible threads connecting personal choices to collective outcomes.

**This is NOT a judgment tool.** It's an educational lens for understanding systemic patterns through cause-and-effect exploration.

## ✨ Features

### 🧪 Normalization Simulator
Simulate the long-term collective impact of everyday financial behaviors. Enter a behavior (or choose from presets) and see:
- **Individual Impact** – Personal consequences over time
- **Collective Impact** – What happens when millions do the same
- **Pressure Points** – Systems that get strained
- **Lever for Change** – Actionable small shifts

### 🔮 Perspective Shift
Explore hypothetical scenarios through multiple-choice thought experiments. Each choice reveals different potential outcomes, encouraging systems-level thinking.

### 📊 Public Balance Sheet
A real-time visualization of global financial metrics featuring:
- Animated global statistics
- Interactive world financial map
- Visual balance bars for economic indicators
- Trend analysis and insights

### 💬 The Archivist (AI Chat)
An AI-powered conversational guide that helps users explore financial behaviors, understand systemic connections, and discover patterns in economic systems.

### 📡 Daily Signal
A rotating daily insight featuring:
- Financial insight of the day
- Normalized behavior spotlight
- Thought-provoking questions

### 🌌 3D System Visualization
An immersive Three.js-powered visualization representing interconnected economic systems with dynamic particle effects and orbital animations.

### 🎨 Additional Features
- **Theme Toggle** – Light/Dark mode support
- **Ambient Sound** – Optional atmospheric audio
- **Keyboard Shortcuts** – Power user navigation
- **Command Palette** – Quick access to all features
- **Live Society Pulse** – Real-time animated indicators
- **Guided Tour** – Interactive onboarding for new users
